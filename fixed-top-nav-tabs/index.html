Fixed Top Nav Tabs

This fixed top navigation is built with HTML, CSS, and vanilla JavaScript. It is fully responsive with a hidden menu on small screens that extends from the left to cover the entire viewport. On large screens, the navbar has a max-width and borders separating links, giving the feeling of "floating tabs".

Just for fun, I added a CSS design element for the logo, which includes animation on hover.
