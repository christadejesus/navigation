3-Tier Responsive Top Navigation

first

This multi-level top navbar is built with HTML and CSS. For full functionality of search and user specific features, additional programming would need to be added.

The top tier would work great for featuring promotions, updates, and calls to action. The middle tier holds the logo, search bar/icon, and user buttons, all a bit different according to screen size. The third tier contains bookmark links that will scroll to areas within this single page when clicked.
